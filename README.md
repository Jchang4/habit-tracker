This is **Amor Habit Tracker**

## Getting Started

Install packages:

```bash
npm i
```

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deploying

Create a new Fly.io App:

```bash
fly launch
```

Update the app using:

```bash
fly deploy
```
